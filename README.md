## 💻 프로젝트 테스트

### clone project

```bash
$ git clone git@github.com:MINI-TEAM3/client.git
$ git clone git@github.com:MINI-TEAM3/admin.git
```

### go to project

```bash
$ cd client
$ cd admin
```

### install npm

```bash
$ npm install
```

### start project

```bash
$ npm run dev
```
